# Timer Sound Assets

This directory should contain the following audio files for the Pomodoro timer:

- **tick.mp3** — Short metronome tick sound (~0.5 seconds, CC0 license)
- **bell.mp3** — Classic timer completion bell (~3 seconds, CC0 license)
- **break-start.mp3** — Soft notification chime for break start (~2 seconds, CC0 license)

## Recommended Sources (Free, Royalty-Free)

- [Freesound.org](https://freesound.org) — CC0 licensed sounds
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/) — Free for commercial use

## Fallback

If these audio files are not present, the app will automatically fall back to
device vibration patterns for tactile feedback. The app will never crash due to
missing audio files.
